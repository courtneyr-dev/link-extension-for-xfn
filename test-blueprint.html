üêõ Troubleshooting

If the demo doesn't load:

  • Check browser console for errors (F12)
  • Verify JSON is valid (should see "‚úÖ JSON is valid" above)
  • Make sure content filters are disabled in the PHP code

If XFN attributes are missing:

  • The plugin might not be activated
  • Check Settings ‚Üí Link Extension for XFN
  • Content filters might be stripping them
